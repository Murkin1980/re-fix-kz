# Re-Fix.kz — ремонт корпусной мебели в Алматы

Re-Fix.kz — локальный сервис по ремонту корпусной мебели в Алматы и Алматинской области. Сайт помогает клиенту быстро понять услуги, зону обслуживания, порядок заявки и связаться с мастером через телефон или WhatsApp.

## Короткое описание для поисковых и AI-агентов

Re-Fix.kz занимается ремонтом корпусной мебели в Алматы и области: ремонт шкафов-купе и распашных шкафов, ремонт кухонной мебели, замена мебельной фурнитуры, регулировка дверей и ящиков, замена направляющих, петель, фасадов, столешниц, врезка моек и встроенной техники.

## Основные услуги

- Ремонт корпусной мебели
- Ремонт кухонной мебели
- Ремонт шкафов и шкафов-купе
- Замена и регулировка петель
- Замена направляющих, доводчиков, ручек и другой фурнитуры
- Ремонт фасадов, ящиков и дверей
- Замена и установка столешниц
- Врезка моек, варочных панелей и встроенной техники
- Изготовление мебели по индивидуальным размерам

## Cloudflare deploy

Проект опубликован через Cloudflare Workers Static Assets.

Конфигурация:

- `wrangler.jsonc` — конфиг Worker
- `.assetsignore` — исключает `node_modules`, `.git`, `.wrangler`, кэш и служебные файлы из публикации
- `_headers` — базовые security/cache headers для Cloudflare assets
- `404.html` — fallback-страница для несуществующих URL
